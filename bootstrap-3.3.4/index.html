ESPN

New ESPN app

Software Dev. Intern ...

Developed an optimized data transfer methodology by moving data processing from mobile devices to ESPN servers. This project was developed in the dynamic programming language Groovy. Entering the internship I had little knowledge of Groovy or the original protocol that I transferred from (an in-house language used by ESPN to process JSON).

The result was a program built off the SpringBoot framework and spanned over 5,000 lines of code (over 10,000 with docs/bundled config json/etc) in 10 weeks. The project, named Dark.Binder was inherited by a development team for performance testing and a goal to entirely replace the existing legacy code that runs the backend of the ESPN app and other mobile applicaitons.

Emma5

The Robot

Part of Power Distribution Unit

More Robot

Independent coursework ...

Contributed to an autonomous robot, named Emma5, that was programmed using Arduino. The robot is powered by a 12 volt car battery and contains 8 infrared sensors to navigate it's surroundings. We developed the power distribution unit to prevent short circuiting which had occurred in the past.

I was advised by Charles 'Chuck' Malloch, PhD and worked closely with peer Adam Siegel. This project was undertaken my first semester at UMass Amherst. Some cool gadgets I got my hands on include EagleCAD, thermistors (heat resistors), multimeters, oscilliscopes, and too much sodder.

We also hacked an old Nintendo-64 steering wheel and gas pedals to steer the robot manually using a wireless arduino module. I have since left the project but the Electrical Engineering lab at M5 continues to add functionality.

First Tech Challenge

Team Member ...

2012 began my defining experience in engineering and computer science through the First Tech Challenge. We designed a 24"x24" robot that could lift objects 10 feet in the air to score points for tournament play. Our team won the Bridgewater State Qualifier and moved onto the State Championship where we placed (a disappointing) second. The robot was programmed in Carnegie Mellon's ROBOTC and was completely fabricated in the high schools machine shop.

After winning at Bridgewater, we had 5 weeks before the State Championship so we made the decision to started from scratch in an attempt to qualify for nationals. We were a single spot out of nationals at the State Championship
